# Placement Quest - Site Beautification PRD

## Overview
Transform the minimal Placement Quest homepage into a professional, modern news/blog site with beautiful design and user experience.

## Current State
- Working Turborepo monorepo with Astro SSR + Vercel deployment
- Homepage displays "Placement Quest" title and "Recent Articles" grid (7-8 articles)
- Articles fetching from Neon database with working /{slug} pages
- Basic Tailwind styling, responsive layout
- Clean codebase with @quest/db, @quest/seo, @quest/ui shared packages

## Goals
Beautify the site to create a professional, engaging user experience for placement industry news that rivals top financial news sites.

1. Enhance hero section with gradient background and better typography
2. Improve article cards with shadows, hover effects, and better images

### Phase 3: Article Pages
11. Enhance article detail page typography and layout

13. Implement related articles section
14. Add social share buttons (Twitter, LinkedIn, Email)
15. Improve code block and quote styling

### Phase 4: Advanced Features
16. Add dark/light mode toggle
17. Implement search functionality
18. Add category filtering

20. Add newsletter signup form

### Phase 5: Performance & SEO

25. Performance optimization (<2s page load)

## Design Inspiration
- Financial Times, Bloomberg, TechCrunch
- Clean, professional, data-driven aesthetic
- Emphasis on readability and content hierarchy
- Subtle animations, not flashy

## Success Criteria
- Professional design that competes with top news sites
- Fast page load times (<2s)
- Mobile responsive on all devices
- Accessible (WCAG AA)
- SEO optimized for Google News

## Technical Constraints
- Must work with existing Turborepo structure
- Keep minimal dependencies
- No heavy frameworks (stay with Astro + Tailwind)
- Maintain SSR for SEO
- Vercel serverless function limits
