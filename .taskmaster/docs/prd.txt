# Universal Quest Template v1.0 - Front-End Beautification PRD

## Overview
Transform turbo-clean into a production-ready, universal template for all Quest sites (placement.quest, relocation.quest, mba.quest, etc.) with Bloomberg/FT-level professional design using DaisyUI + Tailwind CSS.

## Current State
- Working Turborepo with Astro SSR + Vercel deployment
- Homepage (index.astro) displays articles from Neon database
- Article detail page ([slug].astro) with basic styling
- Minimal Tailwind CSS styling
- BaseLayout with barebone HTML structure
- @quest/db package handles database queries (working, don't touch)
- @quest/ui package for shared components (currently minimal)

## Goals
Create a beautiful, professional, universal template that:
- Works for all Quest sites with zero configuration
- Matches placement.quest reference design quality
- Uses DaisyUI for consistent, themeable components
- Supports per-site branding via CSS variables
- Includes hero video/image system
- Professional blog/articles layout
- Fast, accessible, SEO-optimized

## Technology Stack
- **Framework**: Astro SSR
- **Styling**: Tailwind CSS v4
- **Component Library**: DaisyUI v5 (zero-JS, CSS-only)
- **Fonts**: Inter Variable
- **Animations**: CSS transforms + Intersection Observer
- **Deployment**: Vercel

## Design System

### DaisyUI Theme Configuration
- Use DaisyUI's CSS variable system for per-site theming
- Create 3 theme variants: financial, relocation, education
- Each site configures via THEME_VARIANT env variable
- DaisyUI provides: dark mode, professional components, consistent styling

### Color Themes
**Financial (placement.quest):**
- Primary: Blue (#0161ef)
- Secondary: Dark Blue (#0154cf)
- Accent: Purple (#6d28d9)

**Relocation (relocation.quest):**
- Primary: Teal (#14b8a6)
- Secondary: Cyan (#06b6d4)
- Accent: Emerald (#10b981)

**Education (mba.quest):**
- Primary: Purple (#8b5cf6)
- Secondary: Indigo (#6366f1)
- Accent: Orange (#f97316)

### Typography
- Font Family: Inter Variable (sans-serif)
- Headings: Bold, large scale (h1: 3xl-5xl, h2: 2xl-3xl)
- Body: 16px base, 1.6 line height
- Professional hierarchy throughout

### Hero System
- Support for video OR static image
- Configuration priority:
  1. HERO_VIDEO_URL (per-app override)
  2. HERO_IMAGE_URL (per-app override)
  3. Default theme-based image from Pexels/Unsplash
- Fallback: Animated gradient matching theme
- Overlay with site title + tagline
- Responsive: video desktop, image mobile

### Animation Strategy (Hybrid)
- **Subtle**: Article cards, page transitions, navigation (FT/Bloomberg style)
- **Dynamic**: Hero section, featured content (Layrid inspiration)
- All animations use CSS transforms (performance)
- Intersection Observer for scroll-triggered effects

## Configuration Structure

### Environment Variables (per app)
```
# Required
DATABASE_URL="postgresql://..."
SITE_NAME="Placement Quest"
SITE_TAGLINE="Navigate the evolving landscape"

# Theme Configuration
THEME_VARIANT="financial"  # or "relocation", "education"

# Optional Customization
PRIMARY_COLOR="#0161ef"     # Override theme primary
SECONDARY_COLOR="#0154cf"   # Override theme secondary
ACCENT_COLOR="#6d28d9"      # Override theme accent

# Hero Configuration (optional)
HERO_VIDEO_URL="https://..."    # Video URL (YouTube, Vimeo, or direct)
HERO_IMAGE_URL="https://..."    # Or static image URL
```

### Default Theme Mappings
- **financial**: Blue corporate palette, boardroom/office imagery
- **relocation**: Teal/green palette, city/moving imagery
- **education**: Purple/orange palette, campus/graduation imagery
